## [0.0.1] - 2026-01-24

### Added

- Initial implementation of pulse-directory MCP server
- `list_servers` tool for browsing MCP servers with search and pagination
- `get_server` tool for retrieving detailed server information by name and version
- PulseMCP API client with authentication support
- Comprehensive test suite (functional and integration tests)
- Full documentation with setup instructions and usage examples
