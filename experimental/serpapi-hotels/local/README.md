# SerpAPI Hotels MCP Server (Local)

Local implementation using stdio transport.

## Usage

```bash
npm install
npm run build
npm start
```

## Development

```bash
npm run dev  # Auto-rebuilds shared module
```

## Configuration

### Environment Variables

| Variable          | Required | Description          |
| ----------------- | -------- | -------------------- |
| `SERPAPI_API_KEY` | Yes      | Your SerpAPI API key |
