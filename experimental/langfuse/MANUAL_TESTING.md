## Test Execution

### Prerequisites

1. **Commit your changes BEFORE running tests**
2. **Set up API credentials** in your `.env` file:

   ```bash
   cp .env.example .env
   # Edit .env with your Langfuse credentials
   ```

### First-Time Setup

```bash
npm run test:manual:setup
```

### Running Tests

```bash
npm run test:manual
```
