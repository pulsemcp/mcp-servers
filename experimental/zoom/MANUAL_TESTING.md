# Manual Testing

## Prerequisites

- Zoom OAuth access token with `meeting:read` and `recording:read` scopes
- `.env` file with `ZOOM_ACCESS_TOKEN` set

## First-Time Setup

```bash
npm run test:manual:setup
```

## Running Tests

```bash
npm run test:manual
```
